Structure of the HTML template

DOCTYPE

The first line in the HTML structure tells us what is the version of HTML that we are using

HTML TAG

An HTML tag tells the browser that everything in between the opening and closing tag is going to be HTML code, consisting of a head and a body.

HEAD TAG

The head is the part of the HTML files that holds information about the webpage and it tells the browser how it should handle the page, for example, we might have a title tag in the head that tells the browser what is the title of this particular document or this particular page.

BODY TAG

It displays the content of a website element containing all the contents of a HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
